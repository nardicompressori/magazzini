⚙ Setup Firebase — Dati Condivisi
Come abilitare la sincronizzazione tra tutti gli utenti:

1. Vai su console.firebase.google.com → Crea progetto
2. Build → Realtime Database → Crea database (modalità test)
3. In Regole, incolla:
{ "rules": { ".read": true, ".write": true } }
4. Copia la URL del database
   (es: https://nardi-xxx-rtdb.firebaseio.com)
5. Apri magazzino_scorrevole.html con un editor
6. Trova: const FIREBASE_URL = ""
7. Sostituisci con: const FIREBASE_URL = "https://tua-url..."
8. Salva e carica su GitHub

Piano gratuito Firebase: 1 GB dati, 10 GB/mese traffico. Più che sufficiente.
